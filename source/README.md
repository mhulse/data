[![Built with Grunt](https://cdn.gruntjs.com/builtwith.png)](http://gruntjs.com/)

```bash
# Install dependencies:
$ npm install
# Update dependencies:
$ npm update
# Development build:
$ grunt
# ... or:
$ grunt dev
# ... or:
$ npm run dev
# Production build:
$ grunt prod
# ... or:
$ npm run prod
```
